### 👋 Hey, I'm Shuvo

I'm a **DevOps Engineer** by day — wiring up container clusters, writing pipelines that don't wake me at 3am, and shipping ML APIs to production. I've been writing code since I was a teenager and never really stopped.

I live in the terminal. My morning starts with `kubectl`, my afternoon involves `terraform plan`, and my evenings are often spent building things nobody asked for but felt necessary. Most of my serious work is in private repos — what you see here is just the surface.

### 🚀 Termi — my most serious public project
> A self-hosted server management platform I built because I was tired of juggling separate SSH, RDP, and VNC clients.
>
> | What it does | How |
> |---|---|
> | Remote access | SSH · SCP · RDP · VNC · local terminal — all from the browser |
> | Security | AES-256-GCM encrypted credentials · Argon2id hashing · TOTP + WebAuthn 2FA |
> | Auth | Google OAuth · session management · role-based access control |
> | Monitoring | Real-time CPU · RAM · Disk · Network per server |
> | Clients | PWA (browser) + native Electron desktop app |
> | Stack | TypeScript · Next.js · Node.js · PostgreSQL · Docker |
>
> → **[github.com/shuvoooo/termi](https://github.com/shuvoooo/termi)**
